# 🔒 403 Forbidden – Information Page Content

👉 403 Forbidden error tab aata hai jab server request ko samajh leta hai, lekin access allow nahi karta. Iska matlab page ya resource exist karta hai, lekin user ke paas usko dekhne ki permission nahi hoti.

## 🧠 Kab hota hai 403 error?

🔐 Jab aap login nahi hote ya authentication required hoti hai  
🚫 Jab aapke account ke paas access permission nahi hoti  
🛑 Jab server security reasons ki wajah se access block karta hai  
📁 Jab protected files ya restricted pages open karne ki koshish hoti hai
